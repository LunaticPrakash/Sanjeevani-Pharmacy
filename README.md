# Sanjeevani Pharmacy

Sanjeevani Pharmacy is an all-in-one pharmacy management system designed to simplify daily pharmacy operations like billing, stock tracking, expiry monitoring, credit management, and profit tracking. It also provides useful suggestions such as better-profit medicine options, intelligent demand predictions, dead stock alerts, and generic or health alternatives (like food choices, yoga, etc.) to help pharmacies reduce losses and make better decisions.

This system is designed especially for Indian pharmacies and supports offline billing, multi-branch management, supplier comparison, and WhatsApp communication.
